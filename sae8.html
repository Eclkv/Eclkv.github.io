SAE - Mettre en place une solution informatique pour l'entreprise

Premier projet de développement web complet : création d'un site de gestion de ludothèque avec base de données, permettant aux utilisateurs de consulter, commenter et noter les jeux.

Django MariaDB HTML/CSS Linux Git

Contexte et Objectifs

Cette SAE nous permet de découvrir le développement web à travers un projet concret : créer un site pour gérer une ludothèque. Les utilisateurs peuvent s'inscrire, consulter les jeux disponibles, les noter et les commenter.

🎯 Mission

Développer en équipe de 3 personnes un site web fonctionnel avec une base de données, hébergé sur une machine virtuelle Linux. Le site doit permettre de gérer des jeux, des utilisateurs et leurs commentaires.

Schéma de Données

Nous devons implémenter les tables suivantes dans notre base de données :

📂 Catégories : id, nom, descriptif
🎲 Jeux : id, titre, année, photo, éditeur, auteur, catégorie
✍️ Auteurs : id, nom, prénom, âge, photo
👤 Joueurs : id, nom, prénom, email, mot de passe, type
💬 Commentaires : jeu, joueur, note, commentaire, date
📋 Listes personnelles : jeux favoris des utilisateurs

Chaque utilisateur peut être soit "professionnel" soit "particulier", ce qui permet de calculer des moyennes de notes séparées pour chaque type d'utilisateur.

Fonctionnalités Développées

👥 Inscription et Connexion

Les nouveaux utilisateurs peuvent créer un compte et se connecter. Distinction entre professionnels et particuliers.

🎲 Catalogue de Jeux

Affichage de tous les jeux avec leurs informations : titre, année, éditeur, auteur et catégorie.

⭐ Système de Notes

Chaque utilisateur peut noter et commenter les jeux. Calcul automatique de la moyenne par type d'utilisateur.

📁 Import de Jeux

Possibilité d'ajouter des jeux en masse via un fichier. Création automatique des auteurs s'ils n'existent pas.

Ma contribution : Responsable de la partie base de données et des fonctionnalités de commentaires.

Fonctionnalités : CRUD complet pour toutes les données, système de notes et commentaires, pages de profil utilisateur.

🎤 Présentation Groupe

Démonstration : 8 minutes de présentation du projet avec démonstration en live.

📱 Dépôt GitHub

Project Summary (English)

🌍 SAE - IT Solution Implementation for Enterprise

Duration: 60 hours over 8 weeks | Team: 3 students | Semester: 2nd

This project was our first complete web development experience, where we created a board game library management website. The application allows users to register, browse available games, rate them, and leave comments.

Technical Stack: Django (Python), MariaDB database, HTML/CSS, Linux Ubuntu server, Git version control

Key Features: User registration system with professional/individual distinction, comprehensive game catalog, rating and commenting system with separate averages per user type, bulk game import functionality, and personal game lists.

My Role: Responsible for database design and implementation, comment system development, and technical presentation. This project provided hands-on experience with full-stack web development, database management, team collaboration, and project planning using Gantt charts.

Outcome: Successfully delivered a functional web application with complete CRUD operations, comprehensive documentation, and live demonstration. This experience sparked my interest in web development and laid the foundation for more advanced projects.
